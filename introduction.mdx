Fume is currently in public beta. Therefore, the only available integrations are as follows:
* Slack
* Github
* GitLab

You can assign tasks right in your Slack and Fume will submit relevant code changes to your git repository in the form of a PR/MR. Fume can ask you questions thourgout to task to make sure the changes are correct.
